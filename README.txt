Instrucțiuni pentru rularea programului:

1. Porniți XAMPP-ul:
   - Deschideți XAMPP Control Panel.
   - Faceți clic pe butonul "Start" pentru serviciile MySQL și Apache.

2. Conectați-vă la baza de date MySQL:
   - Accesati butonul "Admin" pentru serviciul MySQL.
   - Autentificați-vă cu credențialele de bază de date sau creati o baza de date noua denumita "watchshopdb", tabelele se vor crea automat la rularea aplicației

3. Rulați fișierul app.py:
   - Deschideți terminalul sau linia de comandă.
   - Navigați la directorul în care se află fișierul app.py.
   - Rulați comanda `python app.py`.

4. Accesați aplicația:
   - Deschideți un browser web.
   - Accesați http://localhost:port (portul este portul pe care rulează serverul Flask, de obicei 5000).

5. Opriți XAMPP-ul:
   - După ce ați terminat de folosit aplicația, opriți serviciile MySQL și Apache din XAMPP Control Panel.
